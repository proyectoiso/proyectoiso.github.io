Proyecto y programas utilizados

Para llevar adelante la creación de una distro/respin de un Sistema Operativo Linux, primero estudiamos un poco de la arquitectura de linux y cómo está compuesta. Luego se identifico qué objetivo se iba a cubrir con esta distribución y luego se investigó los programas que fueran compatibles con el sistema operativo actual para generar una imagen ISO del mismo. Todo esto en el marco de la cátedra de Sistemas Operativos impartida por el profesor Javier Ruiz Diaz en la Universidad de la Cuenca del Plata dentro del segundo año de la carrera de Sistemas de Información.


Aquí una lista de los posibles programas para generar una imagen ISO:


  • Systemback

    Es una herramienta de restauración del sistema para distribuciones de Linux. Permite crear puntos de restauración y copias de seguridad de todo el sistema operativo, restaurar a estados anteriores y crear un archivo ISO del sistema operativo actual para su reinstalación o distribución. En particular, esta fue la herramienta que me sirvió a mí.

  • ISO-Snapshot

    Permite generar una imagen ISO del sistema operativo actual de una distribución Linux, similar a cómo era en el momento en que se creó el snapshot. Esta imagen ISO puede ser usada para crear una instalación personalizada del sistema operativo o para realizar respaldos completos.

  • Remastersys

    Era una herramienta popular en Linux que permitía a los usuarios crear una copia de su sistema operativo en un archivo ISO. Podías usar este ISO para reinstalar el sistema en otra máquina o distribuir tu configuración personalizada. Aunque el proyecto original ya no está activo, sigue siendo una referencia importante para aquellos que quieren crear distribuciones personalizadas de Linux.


Modo de uso de Systemback

Después de instalarlo, inicie la aplicación Systemback desde el Tablero o el Menú.
La interfaz principal predeterminada de Systemback se verá como se muestra en la siguiente imagen. Debe presionar donde dice Crear Sistema Live.

Como siguiente paso podremos especificar el nombre deseado (opcional) para la imagen, también se nos brinda la posibilidad de marcar la casilla para incluir las configuraciones de usuario; luego presionamos en Crear Nuevo.

Ahora simplemente resta esperar a que el proceso termine.

Una vez finalizado el proceso, la aplicación nos devuelve una pantalla donde podemos apreciar un comprimido y su peso en Gb , lo que nos permitirá evaluar, si es apta para ser convertida en formato ISO instalable o solo podremos grabarla en una usb para su uso en vivo. Como siguiente paso presionamos en Convertir a ISO.

Luego de terminar las operaciones, obtendremos la imagen ISO que se ubicara en /home desde donde podremos disponer de ella.
